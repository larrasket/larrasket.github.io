Solving Introduction to Statistical Learning with Applications

Solutions for An Introduction to Statistical Learning, in Go and Julia.

After playing for a while with (Norvig, 1991) it got me interested in learning more about statistical learning, one of the most common frameworks for machine learning. (James et al., 2023) gives a very thorough introduction with application using the Python programming language (yes, the book name is pretty descriptive).

While I enjoy the book, I rarely enjoy writing Python. Here I provide my notes and exercise solutions for the book using both the Go and Julia programming languages.
