Web is one of the biggest thing that was done wrong in the software industry, in my opinion. For now, there is no and there is not going to be a web browser that is capable of being 'proper'. I'm using ungoolged-chromium. This page contains some of the extensions that makes the web less painful to me.

I'm currently migrating (again) to ungoogled-chromium, saying goodbye to Brave's nonsense. Ans as usual I've to move all my web extensions; my luggage. I told myself why not to do while writing them down here, I might not be able to remember all of them once anyway.

It'd be sad if most of those extensions would be purged due to Google deduction on Manifest V2.

  • No History; There is no need to this fucking annoying feature. I do not want to store history, If I want to remember something I'd bookmark it.
  • Wayback Machine; Because I never trust the state of web.
  • 4chanX; g-kun?
  • Bitwarden; I basically use pass, but I keep useless accounts in a useless password manager.
  • Disable Extensions Temporarily; To make sure non of your luggage is the reason that some fucking webpage does not work.
  • Feedbro; RSS reader, with private config.
  • Storage Area Explorer; To export foxyproxy.
  • FoxyProxy Standard; No enough time to open tor nor to change proxy. With private config.
  • Georgify; Readable HN.
  • GoFullPage; Page capture.
  • Image search for Yandex; Russians have better image search functionalities.
  • Linux Scroll Speed Fix; Scroll sucks on Linux.
  • RES; Handy when you are having conversation with Americans.
  • Return YouTube Dislike; Hell yeah.
  • Simple Translate; Pocket Google translate. With private config.
  • Stylus; Make all the web use Georgia fonts. With private config.
  • Tampermonkey; The web is bloated; let's bloat it even more! With private config.
  • uBlacklist; To block Quora from search results. private config.
  • Vimium; Should come preinstalled. private config
  • Prettier Lichess; Lichess is ugly. private config
